# Decision Trees and Random Forests

## Objective
The objective of this task is to learn and implement tree-based machine learning models for classification. This project demonstrates the use of Decision Trees and Random Forests on the Iris Dataset, along with model evaluation, overfitting analysis, feature importance interpretation, and cross-validation.

---

## Dataset
Dataset Used: Iris Dataset

Source:
Scikit-learn Built-in Dataset

The Iris dataset contains 150 samples of iris flowers belonging to three species:
- Setosa
- Versicolor
- Virginica

Features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target:
- Flower Species
